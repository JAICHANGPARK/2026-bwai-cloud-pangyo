# Build with AI Cloud x Pangyo 2026

Google GDG Cloud Korea가 주최하는 **Build with AI Cloud x Pangyo 2026** 행사의 **Gemma 4 핸즈온 세션** 공식 리포지토리입니다.

## 참가자 사전 준비 가이드
핸즈온 세션에 원활하게 참여하기 위해, **행사 시작 전(5월 31일 일요일 오후 2시 이전)에** 개인 노트북에 실습 도구 설치 및 Gemma 4 모델 다운로드를 완료해 주시기 바랍니다. 세션 진행 중 실시간으로 다운로드하면 대기 시간이 길어질 수 있으므로 사전 다운로드가 꼭 필요합니다.

상세한 설치 및 준비 과정은 아래 문서 링크들을 확인해 주세요.

*   **[전체 사전 준비 안내 (메인 가이드)](./docs/README.md)**
*   **[Ollama 설치 및 모델 다운로드 가이드](./docs/ollama_guide.md)**
*   **[LM Studio 설치 및 GGUF 모델 로드 가이드](./docs/lmstudio_guide.md)**
*   **[llama.cpp 빌드 및 실행 가이드](./docs/llamacpp_guide.md)**
*   **[Gemma 4 모델 및 하드웨어 사양 상세 안내](./docs/gemma4_guide.md)**
